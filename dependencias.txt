Dependências: 
    node -v:
        Verificar se contém o node.

    npm init -y:
        Inicializar o node no arquivo.
    
    npm install express sqlite3 body-parser cors:
        Baixar as bibliotecas.

    node server.js:
        Colocar no ar o servidor.

    npm install nodemailer:
        Biblioteca para facilitar o envio para o email.
    

    Depois inicializar o index.html para capturar as informações.
